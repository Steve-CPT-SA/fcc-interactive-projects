Introduction

HTML and CSS are the backbone of web development.

HTML(HyperText Markup Language) creates the structure and organises content.

CSS (Cascading Style Sheets) equips you with the tools to design, layout and add visual flair.

Together they enable the creation of accessible, interactive and visually engaging websites.

<!DOCTYPE html>

HTML Basics

HTML is used to build a webpage's structure. Key concepts include:

  • Document Structure: Every HTML document starts with a <!DOCTYPE html>declaration, followed by <html>, <head>, and <body>.
  • Elements and Tags: Use tags like <h1> - <h6> for headings, <p> for paragraphs, <a> for links, <img> for images, and <ul> / <ol> with <li> for lists.
  • Semantic HTML: Adding appropriate tags (like <article>, <section>, <nav>) helps outline the meaning and enhance accessibility, aiding search engines and users alike.

CSS Fundamentals

CSS adds style and layout to your HTML structure through:

  • Selectors and Properties: CSS rules are applied to HTML elements using selectors (like element, class, or ID selectors) and apply properties such as color, font-size, and margin. For example:body { color: #2e2e3f; font-size: 14px; margin: 0 auto; }
  • The Box Model: Every element is treated as a box composed of its content, padding, border, and margin.
  • Stylesheet and Specificity: External stylesheets keep content (HTML) and design (CSS) separate and manageable.

Layout and Responsiveness

Modern web design requires flexible layout techniques and reponsiveness:

Below are key concepts to achieve this:

  • Layout Techniques:
    • Flexbox Using display: flex; aligns and distributes space among items in a container.
    • Grid: Using display: grid; creates a two-dimensional layout system suitable for complex designs.
  • Responsivness:
    • Media Queries: (@media) allows CSS rules to adapt based on the viewport's dimensions, ensuring a great user experience on devices both large and small.
    • Relative Units (like em, rem, %, vh, and vw) help in creating designs that scale fluidly across different screen sizes.

Example and Conclusion

HTML Example:

Below is a simple HTML and CSS example that demonstrates key concepts. 


            <!DOCTYPE html>
            <html lang="en">
              <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Example Page</title>
                <link rel="stylesheet" href="styles.css">
              </head>
              <body>
                <header>
                  <h1>Hello, World!</h1>
                </header>
                <main>
                  <p>This is an example of a page using HTML.</p>
                </main>
                <footer>
                  <p>Footer content here.</p>
                </footer>
              </body>
            </html>

CSS Example: 


            /* styles.css example */
            body {
              font-family: Ariel, sans-serif;
              background-color: #fafafa;
              margin: 0;
              padding: 0;
            }

            header{
              background: #333;
              color: #fff;
              padding: 1em;
            }

Conclusion:

Mastering HTML and CSS is essential for creating functional and attractive web experiences. HTML lays the foundation by providing structure and sematic meaning, while CSS brings the design to life with effective styling and layout techniques. By exploring topics like semantic HTML, the box model, flexible layouts (with Flexbox and Grid), and responsive design practices you will develop the skills needed to build professional, accessible websites that look great on any device.
